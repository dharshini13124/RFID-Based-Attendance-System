# RFID Based Attendance System

## Project Overview

This project is an automated attendance monitoring system developed using RFID technology.

Each user is provided with a unique RFID card. When the card is scanned using an RFID reader, the system verifies the card information and records attendance automatically.

## Components Used

- ESP8266 NodeMCU
- RFID Reader (MFRC522)
- RFID Tags
- I2C LCD Display
- Buzzer
- WiFi Connectivity

## Features

- Automatic attendance recording
- LCD display notification
- Buzzer feedback
- Google Sheets integration
- WiFi-based data logging
